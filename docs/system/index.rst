AAS Facility Codes for Rubin Observatory
========================================

When publishing Rubin Observatory results in AAS journals (AJ, ApJ, ApJL, ApJS), authors must include a “Facilities” section identifying the observatories and data archives used in their research.
The following official facility codes should be used when citing Rubin Observatory data and Rubin-hosted data access services.

Official facility codes
=======================

**Rubin:Simonyi** — Simonyi Survey Telescope

**Rubin:USDAC** — U.S. Data Access Center (US DAC)

Codes to include in AAS journals
================================

::

   Facilities: Rubin:Simonyi, Rubin:USDAC
