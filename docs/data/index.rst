Data Preview 0.3
================

- The DP0.3 data set was generated by members of the Rubin Solar System Pipelines and Commissioning teams, with help from the LSST Solar System Science Collaboration, in particular: Pedro Bernardinelli, Jake Kurlander, Joachim Moeyens, Samuel Cornwall, Ari Heinze, Steph Merritt, Lynne Jones, Siegfried Eggl, Meg Schwamb, Grigori Fedorets, and Mario Juric.

Data Preview 0.2
================

- "CosmoDC2: A Synthetic Sky Catalog for Dark Energy Science with LSST", `Korytov et al. (2019) <https://ui.adsabs.harvard.edu/abs/2019ApJS..245...26K/abstract>`_
- "The LSST DESC DC2 Simulated Sky Survey", `LSST DESC (2021) <https://ui.adsabs.harvard.edu/abs/2021ApJS..253...31L/abstract>`_
- "DESC DC2 Data Release Note", `LSST DESC Release Note (2021) <https://ui.adsabs.harvard.edu/abs/2021arXiv210104855L/abstract>`_
- The data set used for DP0.2 is the 300 square deg of simulated, LSST-like images and catalogs generated by the Dark Energy Science Collaboration (DESC) for their `Data Challenge 2 (DC2) <https://arxiv.org/abs/2010.05926>`_.

==================================
Bibtex for technical documentation
==================================

The GitHub repository `lsst-texmf <https://github.com/lsst/lsst-texmf/blob/main/texmf/bibtex/bib/lsst.bib>`_ contains bibtex-style
references for `technical documents <https://rubinobservatory.org/for-scientists/documentation/tech-docs>`_. Please cite Rubin Tech Notes as documents and not as footnote URLs.
