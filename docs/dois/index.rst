Primary Rubin Observatory Dataset DOIs
======================================

**Data Preview 0 (DP0)**
A DOI was not issued for DP0. This preview was based on DESC simulation datasets, which have their own DOIs. Authors should cite the specific simulation DOI if applicable.

**Data Preview 1 (DP1)**

- **DOI:** 10.71929/RUBIN/2570308
- **Title:** Legacy Survey of Space and Time Data Preview 1
- **URL:** https://www.osti.gov/servlets/purl/2570308
- **Year:** 2025


In addition to the primary data release DOIs listed above, Rubin Observatory assigns DOIs to specific dataset types, catalogs, and derived data products.
When a more specific DOI exists for the dataset you used, cite that DOI in preference to the broader data release DOI.
