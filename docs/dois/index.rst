##################################
Digital Objects Identifiers (DOIs)
##################################

Unique DOIs are created for all Rubin instruments, and for each data release and the individual data products within each release.

In addition to the relevant DOIs, publications should include the AAS facility codes:

- ``Rubin:Simonyi``, for the Simonyi Survey Telescope
- ``Rubin:USDAC``, for the U.S. Data Access Center (US DAC)
