####################################
Technical documentation (Tech Notes)
####################################

Rubin staff and collaborators produce series of Rubin "tech notes" that are available via `lsst.io <https://www.lsst.io/>`_.

Please use the bibtex-style references for Rubin tech notes, available in the GitHub repository `lsst-texmf <https://github.com/lsst/lsst-texmf/blob/main/texmf/bibtex/bib/lsst.bib>`_, instead of citing them as URLs in footnotes.

See also this `technical documention webpage <https://rubinobservatory.org/for-scientists/documentation/tech-docs>`_ and this `curated list <https://ui.adsabs.harvard.edu/public-libraries/oue8xvvpTjqeYPvR5YW6VA>`_ of Rubin technical publications.
